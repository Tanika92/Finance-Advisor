# 💰 AI-Powered Personal Finance Advisor

An AI-powered Personal Finance Advisor built with **Python** and **Machine Learning** that helps users analyze expenses, evaluate financial health, predict future spending, and receive personalized financial advice.

## 🚀 Features

- Track monthly income, expenses, and savings
- Expense analysis by category
- Financial health assessment
- Budget comparison and savings insights
- Expense visualization using Bar & Pie charts
- Expense prediction using Random Forest Regressor
- Personalized financial recommendations

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📂 Dataset

The project uses a personal expense dataset containing:
- Date
- Category
- Amount
- Payment Mode
- Month

## 🤖 Machine Learning

- **Model:** Random Forest Regressor
- **Features:** Category, Payment Mode, Month
- **Target:** Expense Amount
